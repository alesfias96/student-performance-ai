# Report Example (struttura)

Il report HTML contiene:

1. **Overview**
   - accuracy totale
   - tempo medio
   - livello complessivo

2. **Topic breakdown**
   - bar chart accuracies per topic
   - radar chart (opzionale)

3. **Error analysis**
   - heatmap (topic × error_type)

4. **Recommendations**
   - elenco priorità (1 alta, 2 media, 3 bassa)
   - motivazione + azioni pratiche

> Nota: i report sono in `reports/` e includono immagini base64, quindi sono facilmente condivisibili.
